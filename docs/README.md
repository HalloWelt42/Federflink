# Federflink - Dokumentation

Mehrteilige Dokumentation. Sie ist zugleich Anleitung fuer Menschen und
Integrationsvertrag fuer Programme und andere KI-Modelle, die Federflink in
eigene Anwendungen einbinden.

1. [00_UEBERBLICK.md](00_UEBERBLICK.md) - Zweck, Leitgedanken, Schichten
2. [01_ARCHITEKTUR.md](01_ARCHITEKTUR.md) - Komponenten, Datenfluss, Degradation
3. [02_API.md](02_API.md) - vollstaendige HTTP-API (Endpunkte, Schemas, Beispiele)
4. [03_ENGINES.md](03_ENGINES.md) - eigene Pruef-/Ergaenzungs-Engine hinzufuegen
5. [04_LERNEN.md](04_LERNEN.md) - Woerterbuch, N-Gramm, Umfeld-Kontext
6. [05_PROFILE_UND_KONFIG.md](05_PROFILE_UND_KONFIG.md) - Profile und Einstellungen
7. [06_BROWSER_PLUGIN.md](06_BROWSER_PLUGIN.md) - Aufbau der Erweiterung
8. [07_CLIENT_INTEGRATION.md](07_CLIENT_INTEGRATION.md) - in eigene Apps einbinden
9. [08_DEPLOYMENT_PI.md](08_DEPLOYMENT_PI.md) - Betrieb auf dem Pi5
10. [09_MODELLE.md](09_MODELLE.md) - empfohlene kleine Modelle

Der Integrationsvertrag steht in [02_API.md](02_API.md), das Rezept zum Einbinden
in eigene Programme in [07_CLIENT_INTEGRATION.md](07_CLIENT_INTEGRATION.md).
